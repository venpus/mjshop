## 5. 현재 프로젝트(mjshop) 적용 시 권장

- **1차 추천:** **Google Gemini 1.5 Flash (무료)**  
  - 무료 한도로 PoC·소규모 운영 가능  
  - 한도 초과 시에도 Gemini 2.0 Flash로 저렴하게 확장  

- **2차 추천:** **OpenAI gpt-4o-mini**  
  - Function Calling으로 SQL 생성·데이터 검색에 적합  
  - 월 수만 원 수준까지면 부담 적음  

- **비용·다국어 추천:** **QWEN (통의천문)**  
  - **Qwen-Turbo:** 출력 단가 최저($0.20/1M), 무료 100만 토큰(90일)  
  - **Qwen-Flash:** Gemini 2.0 Flash와 동일 단가, 무료 티어 동일  
  - 한국어·중국어 지원이 좋아 중국 결제·창고 등 다국어 데이터 검색에 적합  

- **비용 절감 팁**
  - 스키마/도움말 등 반복 전달은 **캐싱** 사용 (Anthropic Prompt Caching, OpenAI Cached input)
  - 배치·비실시간 분석은 **Batch API** 사용 시 50% 할인 (OpenAI, Google, Anthropic, QWEN)
  - 짧은 대화·단순 검색은 **미니/나노 모델** 또는 **QWEN Qwen-Turbo** 사용
